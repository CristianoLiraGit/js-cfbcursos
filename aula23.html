Aula 23 - Parâmetros rest

Quando eu não quero especificar a quantidade de parâmetros que vou passar para dentro da função, uso os parâmetros rest. Exemplo:

        function soma(n1, n2) {
            return n1+n2
        }
        
        console.log(soma(10, 5))
