Aula 24 - Funções Anônimas

  • Não têm nome padrão
  • Não são salvas na memória
  • Somente serão criadas no momento da execução
  • Deve ser associada a uma variável

Exemplo:

        let f = function(v1, v2){return v1 + v2}

        Obs.:  o código acima pode ser feito da outra maneira:

        let f = function(v1, v2){
            return v1 + v2
        }

        console.log(f(10, 5 ))

Mesmo código acima, usando parâmetros rest:

        let f = function(...valores){
            let res = 0
            for(v of valores) {
                res += v
            }
            return res
        }

        console.log(f(10, 5 ))

Função Construtor. Exemplo:

        const f = new Function("v1", "v2", "v3", "return v1 + v2 + v3")

        console.log(f(10, 5, 15))

É um tipo de função anônima mais resumida. Note que é necessário usar o F maiúsculo na palavra Function
