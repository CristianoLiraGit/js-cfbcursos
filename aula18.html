Aula 18 - while e do while

    let n = 1

    while (n < 6) {
        console.log('Oi')
        n++
    }

Este é o while tradicional. No do while, o teste lógico é feito depois. Vejamos:

    let n = 1
    do {
        console.log('Oi')
        n++
    } while (n < 6)
