Aula 22 - Funções Parametrizadas

Exemplo 1:

            function soma(p1) {
                console.log(p1)
            }
            
            soma('Cristiano')
            soma(2000)
            soma(5.2)

Posso passar qualquer parâmetro para ser usado na função e retornado na tela.

Exemplo 2:

            function soma(n1, n2) {
                console.log(n1 + n2)
            }
            
            soma('Cristiano')
            soma(2000)
            soma(5.2)

Vai dar undefined em 'Cristiano', pois se trata de string. Em "soma(2000)" e em "soma(5.2)" vai dar NaN porque só foi passado um parâmetro para a função.


Exemplo 3:

            function soma(n1, n2) {
                console.log(n1 + n2)
            }
            
            soma(10, 5)

Vai retornar a soma de 10 e 5 = 15. Caso eu passe apenas um parâmetro ao chamar a função, vai NaN. É necessário estabelecer valores padrões para os parâmetros. Exemplo:

            function soma(n1=0, n2=0) {
                console.log(n1 + n2)
            }
            
            soma(10)

Ao chamar somente 10, o programa vai entender que vai ser uma soma entre 10 e 0, resultando em 10, pois o parâmetro não chamado assumiu o valor padrão de 0.
