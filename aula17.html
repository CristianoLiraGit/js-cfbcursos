Aula 17 - Loop while

Loop while é usado quando não se sabe a quantidade de iterações

        let n = 0

        while(n < 10) {
            console.log(n)
            n++
        }

Um exemplo clássico do loop while é o do fatorial. 5! = 5 x 4 x 3 x 2 x 1 = 120. Para realizar esta operação com JS:

    let n = 5
    let fat = 1

    while(n >= 1) {
        fat *= n
        n--
    }

    console.log(fat)
